word2vec of Dota2 team

Posted on 2017-06-01

see this post

>
Refer to wikipedia, word2vec is a group of related models that are used to produce word embeddings.

If we treat one Dota2 match team as a ‘Dota2 sentence’, for example, ‘mirana skeleton_king lich riki sven’, and each hero name as a ‘word’, so we can use word2vec to model these ‘words’.

Here I use this Dota2 api to fetch Dota2 match data(in this trial there are 40k+ matches, which means I have 80k+ sentences(5 words(heroes) -> 1 sentence)).

After that, I use gensim library to do the ‘word2vec’.

Then, visualize it

Here is result:

red - Carry
green - Support
blue - Other
