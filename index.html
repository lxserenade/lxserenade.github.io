What is service mesh

A service mesh is a configurable, low latency infrastructure layer designed to handle a high volume of network-based interprocess communication amoung application infrastructure services using APIs.

Software

  • Consul
  • Istio
  • etcd

Key feature

  • Service Discovery
  • Health Checking
  • KV Store
  • Secure Service Communication
  • Multi Datacenter
